# 结构体、联合体、枚举类

## 结构体
- 通常用于保存数据，没有操作的类型（类偏操作），且结构体成员共有，方便

- ```c++
  struct Name{
      共有成员;
  protected:
      保护成员；
  private:
      私有成员；
  };
  ```
## 联合体
- 上述代码，struct 换为 union
- union 所有成员公用相同内存单元，所以不会同时存在


## 枚举类
- ```c++
  // eg.
  enum class Type: char { A, B, C, D };
  // used:
  Type::A
  ```

- 不定义低层类型，默认是 int，新标准可以定义低层类型以外的其它类型

- 优势：

  - 强作用域，枚举类内
  - 转换限制，不能与整型隐式互相转换，不同枚举类无法比较
  - 可以指定低层类型
